## Определение
**Linux** — это стандарт операционных систем, построенный на принципах POSIX. Для DevOps это база, так как большинство инструментов (Docker, K8s) работают на нем.

### Структура каталогов
- `/etc` — конфигурационные файлы 
- `/var/log` — системные логи 
- `/bin` — исполняемые файлы 
- `/home` — домашние папки пользователей

### Базовые команды
> - `pwd` — путь к текущей директории.
> - `ls -la` — список файлов с правами доступа.
> - `cd` — смена директории.
> - `mkdir` / `touch` — создание папок/файлов.
